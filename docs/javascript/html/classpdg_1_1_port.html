Draw an elliptical arc using the specified center, radii, and angle range.

This function draws an elliptical arc (a portion of an ellipse) using the specified center point, radii, and angle range. The arc is drawn with the given attributes for styling.

Parameters
centerThe center point of the ellipse in screen coordinates
xRadiusThe horizontal radius of the ellipse in pixels
yRadiusThe vertical radius of the ellipse in pixels
startAngleThe starting angle of the arc in radians (0 = 12 o'clock position, positive = clockwise)
endAngleThe ending angle of the arc in radians
attrsThe Attributes object containing styling information (line color, thickness, fill, etc.)

The arc is drawn as a portion of an ellipse defined by the center point and radii. The angle parameters specify which portion of the ellipse to draw, starting from startAngle and ending at endAngle.

Angles are measured in radians, with 0 radians pointing up (12 o'clock position). Positive angles go clockwise, negative angles go counter-clockwise.

Warning
If startAngle equals endAngle, no arc will be drawn
Note
The arc is drawn using the current transformation matrix and clipping region

◆ drawSphere()

drawSphere ( Point center,
number float radius,
Attributes attrs )

Draws a textured sphere at the specified center point with given radius and styling attributes.

This function renders a 3D sphere with a texture applied from a single Image. The sphere is drawn using perspective projection and includes lighting calculations for realistic 3D appearance.

Parameters
centerThe center point of the sphere in screen coordinates
radiusThe radius of the sphere in pixels
attrsThe attributes to use to draw the sphere

Attributes

  • texture() The Image containing the texture to apply to the sphere
  • sphereRotation() The rotation angle of the sphere around its vertical axis in radians
  • polarOffset() The offset for the polar coordinates (latitude/longitude mapping) in radians
  • lightOffset() The offset for the lighting calculation in radians

The function creates a 3D sphere by mapping the 2D texture from the Image provided by attrs.texture onto a spherical surface. The texture is wrapped around the sphere using spherical coordinates, offset by attrs.sphereRotation, with the rotation happening around the pole define at attrs.polarOffset.

The lighting calculation uses the attrs.lightOffset parameter to determine how light falls on the sphere, creating a realistic 3D appearance with highlights and shadows.

Warning
Very large texture (image) or radius values may impact rendering performance
