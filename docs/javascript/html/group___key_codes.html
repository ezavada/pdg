◆ keyCode_Alt

const keyCode_Alt = 58

platform-specific keycode for the Alt/Option key (generic, may not distinguish left/right)

This constant contains the keycode value for detecting either Alt (Windows/Linux) or Option (Mac) key on the current platform. On Windows, this is the generic VK_MENU code that doesn't distinguish between left and right. On other platforms, it equals keyCode_LeftAlt.

Platform values:**

  • Mac OS X: 58 (same as keyCode_LeftAlt)
  • Windows: 18 (0x12 - VK_MENU, generic alt key)
  • Linux/GLFW: 342 (same as keyCode_LeftAlt)

◆ keyCode_LeftAlt

const keyCode_LeftAlt = 58

platform-specific keycode for the left Alt/Option key

This constant contains the correct keycode value for the left Alt (Windows/Linux) or Option (Mac) key on the current platform. Use this when checking #KeyEvent.keyCode in eventType_KeyDown or eventType_KeyUp event handlers to detect left Alt/Option key presses in a cross-platform way.

Platform values:**

  • Mac OS X: 58 (0x3A - NSEvent hardware keycode for left Option)
  • Windows: 164 (0xA4 - VK_LMENU virtual keycode for left Alt)
  • Linux/GLFW: 342 (GLFW_KEY_LEFT_ALT)
